# Резервное копирование
## О программе:
### Программа создана для резервного копирования фотографий, полученых с сайта "Вконтатке" и загруженных на "Яндекс Диск". Дополнительно возможно сохранить фотографии на копьютер.
## Инструкция:
#### 1. Для работы с программой нужно получить токен(ключь) для работы с сервисом, инструкцию для получения вы найдете по следующим ссылкам:
* Для VK - https://dev.vk.com/api/access-token/implicit-flow-user
* Для Yandex Disk - https://yandex.ru/dev/disk/poligon
#### 2. Полученый токен(ключь) нужно вставить в следующий файл:
* Для VK - token_vk.py (Пример: token_vk = "ваш токен")
* Для Yandex Disk - token_yandex.py (Пример: token_yandex = "ваш токен")

#### 3. Программа спросит ваш id вк, найти его можно в вашем профиле в разделе личные данные, нужный id состоит из 9 цифр.
#### 4. Программа спросит с какого альбома нужно загрузить фотографии, если нужно загрузить только аватарки профиля, ответь "No", если нужно загрузить фотографии с конкретного альбома ответь "Yes", и укажите id альбома, его можно найти в адресной ссылке открытого альбома.
